# Pokédex — Projeto  Completo

Este projeto é uma Pokédex interativa desenvolvida utilizando HTML, CSS e JavaScript puro, consumindo a PokéAPI para exibir Pokémons, seus dados e permitir montar seu time dos sonhos com seus Pokémon favoritos através de um time, onde você pode adicionar anotações acerca de cada Pokémon.

A interface foi inspirada no design da Pokédex, com alterações no layout e utilizando uma imagem de fundo e uma área de tela interativa.

---

#Funcionalidades

 1. Buscar Pokémons
- Busca por Nome ou ID.
- Navegação entre Pokémon através de uma lista.

 2. Listagem de Pokémons
- Lista inicial de 12 Pokémons.
- Botão “Mostrar mais” carrega mais 12 por vez.
- Cada item exibe:
  - Nome  
  - ID  
  - Sprite frontal  

 3. Exibição de Detalhes
Ao selecionar um Pokémon, você vê:
- Tipos  
- Habilidades  
- Altura e peso  
- Sprite animado ou padrão  
- Lista de status base (HP, Ataque, Defesa etc.)  
- Botão para adicionar à equipe ou editar anotação

 4. CRUD de Favoritos (Equipe)
A aba *Equipe* permite:
- Adicionar Pokémon aos favoritos  
- Escrever anotações personalizadas  
- Editar anotações  
- Remover da equipe  
- Tudo salvo no *LocalStorage*

5. Persistência Local
- Toda a equipe é armazenada no navegador
- Toda vez que retornar sua equipe estará lá

6. Tecnologias e linguagens utilizadas
- HTML — Estrutura da página
- CSS — Estilo, animações e layout responsivo
- JavaScript — Lógica, consumo da API 
- PokéAPI — Fonte de dados dos Pokémons
- LocalStorage — Armazenamento de equipe e anotações
